# Line Bot 

使用 Python LINE Bot SDK 在 Heroku 上架設一個簡單的回話機器人，回應內容使用request爬取巴哈姆特手遊哈啦版內容，輸入「嗨」回應全部板塊名稱，輸入「子板塊名稱」顯示最新10篇文章和超連結，輸入其他詞彙自動導向全部主題。
